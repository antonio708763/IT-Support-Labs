# IT-Support-Labs

This repository is my IT support portfolio for documenting hands-on lab work, troubleshooting practice, and technical learning. It is written for employers and reviewers who want a clear view of the systems I have practiced with and the skills I am building.

## Projects

- **VirtualBox Active Directory Lab**: Windows Server and client lab work focused on domain services, user and group management, Group Policy, DNS concepts, and common support tasks.
- **Proxmox Homelab**: Personal virtualization environment used to practice server management, virtual machines, storage, networking, backups, and organized documentation.
- **Action1 Endpoint Management Practice**: Endpoint management practice focused on device inventory, patch management concepts, remote support workflows, reporting, and automation basics.

## Skills Demonstrated

- Active Directory user, group, and policy administration practice
- Windows client and server troubleshooting
- Virtualization with VirtualBox and Proxmox
- Endpoint inventory and patch management concepts
- Basic networking, DNS, and lab environment planning
- Help desk-style documentation and troubleshooting notes
- Security-minded habits such as avoiding published secrets and documenting safe configurations

## Note

This repository contains personal lab work created for learning and portfolio purposes. It does not represent production enterprise administration experience, but it demonstrates practical IT support skills I am actively developing.
