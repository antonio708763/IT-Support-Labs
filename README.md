# IT-Support-Labs

## About This Portfolio

This repository is my personal IT support portfolio. It documents hands-on lab practice with Active Directory, Proxmox, Action1, virtualization, endpoint management, troubleshooting, and technical documentation.

The goal is to show employers how I am building practical IT support skills in a lab environment. This is personal learning and portfolio work, not paid production administration experience.

## Featured Projects

### [VirtualBox Active Directory Lab](Projects/active-directory-lab/README.md)

Windows Server 2022 and Windows 11 lab work focused on Active Directory, domain joining, user and group management, Group Policy basics, DNS concepts, file sharing, and troubleshooting. Screenshots are included and linked from the project page.

### [Proxmox Homelab](Projects/proxmox-homelab/README.md)

Personal Proxmox homelab summary based on my HaaS-platformV2 documentation, including VM templates, ZFS-backed storage concepts, Docker hosting, Traefik, DNS/HTTPS access, and backup/restore workflows. Screenshots are planned and will be added after review and redaction.

### [Action1 Endpoint Management Practice](Projects/action1-endpoint-management/README.md)

Lab notes for practicing Action1 endpoint inventory, remote device management concepts, patch management concepts, software deployment concepts, and endpoint monitoring. Screenshots are planned and will be added after review and redaction.

## Skills Demonstrated

- Active Directory user, group, domain, and Group Policy practice
- Windows Server and Windows client troubleshooting
- Virtualization with Oracle VirtualBox and Proxmox VE
- Basic networking, DNS, NAT, and isolated lab design
- Endpoint inventory, monitoring, and patch management concepts
- Docker and Docker Compose service hosting concepts
- Reverse proxy and DNS/HTTPS service access concepts
- Backup, restore, and documentation habits
- Help desk-style troubleshooting notes
- Security-minded documentation, including avoiding secrets and redacting screenshots

## Current Status

This portfolio is actively being built and refined. The Active Directory lab has the most complete documentation and includes screenshots. The Proxmox homelab and Action1 endpoint management pages are started, with screenshots and deeper walkthroughs planned.

All work in this repository is personal lab work for learning and portfolio purposes. I do not claim that these labs are production enterprise environments, and I avoid documenting secrets, real user data, private credentials, or sensitive system details.
